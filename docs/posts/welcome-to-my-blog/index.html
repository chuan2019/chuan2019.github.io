Welcome to My Technical Blog

After publishing on Medium for a while, I’ve decided to move my technical writing to a self-hosted personal blog. This post is a quick note on why, and what to expect.

Why Move Away from Medium?

Medium is a great platform for reaching readers, but over time a few friction points added up:

  • Paywalls — some readers couldn’t access my articles without a subscription
  • No full control — formatting, code highlighting, and math rendering were limited
  • SEO — I wanted my own domain to build authority around my own name

What This Blog Is

This site is built with Jekyll and the Chirpy theme, hosted on GitHub Pages. It’s fast, free, and fully under my control.

Everything is written in Markdown, version-controlled with Git, and automatically deployed on every push.

What to Expect

I’ll write about:

  • Software architecture and design decisions
  • Machine learning engineering and practical implementations
  • Systems — databases, distributed systems, infrastructure
  • Short-form notes on tools, libraries, and techniques I find useful

If you’re interested in any of these topics, the RSS feed is the best way to follow along.

Thanks for stopping by.
